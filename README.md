## Topics Covered : 

- x-data | x-show | x-cloak attributes
- x-on attribute and events
- x-text attribute
- x-transition attribute
- x-effect attribute
- x-bind attribute
- x-model attribute
- x-if attribute
- x-for attribute
- Manipulating state
- x-ref attribute & $refs
- x-html attribute
- $el
- x-init attribute
- x-init attribute and $watch
- $dispatch
- $data
- Alpine Stores and $store

Also a store example of chocolate store is created
